EventIF

Projeto de organização de um evento fictício em desenvolvimento para estudo das funcionalidades do Python e de seu framework Django. O projeto conta com duas funcionalidades principais: as inscrições e o contato. Na inscrição, o usuário insere seus dados para se inscever como ouvinte no evento, recebendo um e-mail de confirmação logo depois. No contato, o usuário, inscrito ou não, fornece seus dados e a mensagem que deseja enviar, que é guardada no banco de dados. No atual momentos, estou desenvolvendo o retorno da mesagem da equipe para o usuário.
